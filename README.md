# AI Class - 10 Social Science Assistant

## Overview

An AI-powered educational assistant designed to help Class 10 students generate structured and exam-oriented Social Science answers.

## Problem Statement

Many students struggle to write detailed and well-structured answers in Social Science examinations.

## Solution

This project utilizes NotebookLM and prompt engineering techniques to generate high-quality, syllabus-aligned answers that improve learning and examination preparation.

## Features

- Structured answer generation
- Exam-oriented responses
- Prompt optimization
- Educational support for Class 10 students

## Technologies Used

- NotebookLM
- Prompt Engineering
- Artificial Intelligence

## Future Improvements

- Chapter-wise specialization
- Answer evaluation system
- Multi-language support
